HTTPServerRequest

public class HTTPServerRequest: HTTPIncomingMessage, ServerRequest

This class implements the ServerRequest protocol for incoming sockets that are communicating via the HTTP protocol. Most of the implementation is in the HTTPIncomingMessage class which implements logic common to this class and the ClientResponse class.
